# 統合フレームワーク

## 目的
行動変容 × 認知科学 × 哲学 を統合した、再現可能な理論体系を構築する。

## フレームワーク構成（Draft）

```
┌─────────────────────────────────────────────────┐
│              理念層（Philosophy）                │
│   老荘 / 禅 / 武士道 / ストア哲学                  │
├─────────────────────────────────────────────────┤
│              認知層（Cognition）                 │
│   メタ認知 / バイアス認識 / 注意制御               │
├─────────────────────────────────────────────────┤
│              行動層（Behavior）                  │
│   習慣設計 / 環境設計 / フィードバックループ        │
├─────────────────────────────────────────────────┤
│              実践層（Practice）                  │
│   日次ログ / 振り返り / 調整                      │
└─────────────────────────────────────────────────┘
```

## 核心原則
1. **小さく始める** - Comfort Zone +5%
2. **環境を設計する** - 意志力に頼らない
3. **振り返りを習慣化する** - メタ認知の強化
4. **型を持つ** - 日本的哲学の実践
