# Tech × Human Layer キャリア戦略

## 目標
**「人の行動と組織文化に介入できる設計者」** になる。

## 学習領域

### 1. システム設計
- DDD（Domain-Driven Design）
- Event-driven Architecture
- Clean Architecture

### 2. プロダクトマネジメント
- ユーザーリサーチ
- 仮説検証サイクル
- メトリクス設計

### 3. AIシステム
- LLM/エージェント設計
- 倫理的AI実装
- Human-AI Interaction

### 4. 教育工学・学習科学
- Instructional Design
- Learning Experience Design
- Assessment Design

### 5. 人材開発・組織行動
- HRTech
- Organizational Behavior
- Culture Design

## 差別化の本質

```
技術力 × 行動変容理論 × 人間理解
```

この組み合わせは市場での代替可能性が極めて低い。
