# Behavioral Tracker

行動変容理論を実装したパーソナルトラッキングアプリ。

## コンセプト
> 10個作る必要はない。
> "思想を動かすアプリ"を1本、ひたすら育て続ける。

## 機能要件

### Core Features
1. **日次ログ**
   - 習慣トラッキング
   - 運動記録
   - 学習記録

2. **行動分析**
   - AIによるパターン抽出
   - 傾向の可視化
   - 異常検知

3. **目標設定**
   - Comfort Zone +5% モデル
   - 漸進的難易度調整
   - マイルストーン管理

4. **読書・哲学ログ**
   - 読書記録
   - 引用・メモ
   - 思考の蓄積

5. **認知・気分トラッキング**
   - 気分の記録
   - エネルギーレベル
   - 集中度

6. **フィードバックシステム**
   - 理論に基づく洞察
   - 行動変容のヒント
   - 長期的傾向分析
